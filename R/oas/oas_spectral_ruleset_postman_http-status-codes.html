Overview

Finders based on IANA HTTP status code registry to detect use of unassigned, unused or invalid HTTP status codes. The regexes used in http-status-codes-unassigned and http-status-codes-unused have been generated from the registry XML data.

Source code for this ruleset can be found at https://github.com/postman-open-technologies/knowledge-base/tree/main/spectral
