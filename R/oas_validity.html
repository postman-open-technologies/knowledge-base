How many valid OpenAPIs do we have?

  • Out of 15,562 entries, 12,018 (77.2%) are valid and 3,544 (22.8%) are invalid
  • 4,587 out of the 5,876 Swagger (v2.x) entries are valid (78.1%)
  • 7,431 out of the 9,686 OpenAPI (v3.x) entries are valid (76.7%)

Further analysis is in progress to understand what are the common causes of errors.

Recommendations

  • Further investigate and analyze common source of errors

Methodology

Results are based on the count of the isValid boolean property in the resource metadata. This flag is set by the kb_oas_validation.py script, which uses the JSON schemas published under the OAI GitHub project. Validation is performed by the Python jsonschema package. Validation errors are saved in a local file for further analysis.

API

Data for this topic can be found under the /oas/statistics/validity endpoint.
