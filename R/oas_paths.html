Findings

How is the path property used in APIs?

  • The average number of path per API is 11.3, ranging from 1 to 310.
  • Out of the 11,588 APIs with a path, 6,380 (55.1%) have 1-5 paths, 1,845 (15.9%) have 6-10 paths, 1,743 (15%) have 11-20 paths, 1,143 (9.9%) have 21-50 paths, and 477 (0.04116327%) have over 50 paths.
